A bunch of React Component Helpers

React Component Helper adds syntactic sugar to React JS Templates. With inspiration from AngularJS’s ng-hide, ng-show, ng-switch and ng-model, React Utils Helper adds components like <RU.Show />, <RU.Hide />, <RU.Switch /> and Two way binding helpers similar to AngularJS.

Here is the List of Components

  • <RU.Show />
  • <RU.Hide />
  • <RU.Switch />
  • <RU.Repeat />
  • <RU.Input />
  • <RU.Textarea />
  • <RU.Select />
  • <RU.Radio />
  • <RU.Checkbox />
